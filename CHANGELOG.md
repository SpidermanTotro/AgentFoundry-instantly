## [0.1.0] - 2025-12-06

### Added
- Initial release of AgentFoundry-instantly
- `Agent` class for creating AI programming agents
- `Task` class for defining agent tasks
- Comprehensive test suite with 96% code coverage
- Complete development infrastructure:
  - Makefile with automated commands (check, repair, test, lint, format, clean)
  - GitHub Actions CI/CD workflow
  - Code quality tools (flake8, pylint, mypy, black)
  - Development and production dependencies
  - Contributing guidelines
  - MIT License
- Documentation:
  - README with quick start guide
  - CONTRIBUTING guide
  - Project structure documentation
- Test automation script (test_all.sh)

### Features
- Easy-to-use Agent and Task abstractions
- Fast setup and deployment
- Extensible and customizable framework
- Lightweight with minimal dependencies
- Automated code quality checks and repairs
